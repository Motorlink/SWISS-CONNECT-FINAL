# SWISS-CONNECT – Konzeptpapier

> Arbeitsdokument zur Projektdefinition und Gesellschaftsstruktur

---

## 1. Projektname

**SWISS CONNECT** – Digitale B2B-Transportplattform für die Schweiz.

## 2. Vision

Eine kartenbasierte, intuitive Transportplattform nach dem Uber-Prinzip: Aufträge erstellen, Fahrer annehmen, Live-Tracking, automatische Archivierung, Abrechnung und KI-gestützte Verwaltung.

## 3. Beteiligte

|--------|-------|

| **MT Transport GmbH** | Operative Expertise, Finanzierung |

## 4. Gesellschaftsform

- **Stammkapital:** CHF 20.000 (Minimum)
- **Gründungskosten:** ca. CHF 3.000–5.000

## 5. Produktumfang

### 5.1 Kernfunktionen
- Transportauftragsverwaltung (Uber-Logik)
- GPS Live-Tracking
- Automatisches Archiv
- Abrechnungssystem (Swiss21, Bexio)
- Dokumentenmanagement (DMS)
- Zeiterfassung
- KI-gestützte Verwaltung

### 5.2 Fahrer-App
- Kartenbasierter Startscreen
- Heimweg-Modus (Aufträge entlang der Route)
- Pop-up Angebots-Flow mit Countdown

### 5.3 Auftraggeber-Portal
- Karte als Startseite
- Sendung in maximal 5 Schritten erstellen
- Aktive und abgeschlossene Transporte einsehen

## 6. Technische Umsetzung

- Backend: Python 3.12 (FastAPI)
- Frontend: TypeScript (Next.js / React)
- Infrastruktur: Docker / Docker Compose
- Datenbank: PostgreSQL
- Cache: Redis
- Storage: MinIO (S3-kompatibel)
